# 1. CSS 是什么？

CSS (Cascading Style Sheets，层叠样式表），是一种用来为结构化文档（如 HTML 文档或 XML 应用）添加样式（字体、间距和颜色等）的计算机语言，CSS 文件扩展名为.css。

通过 CSS，你可以控制网页的外观和样式，包括颜色、字体、间距、布局等。

# 2. 选择器

属性 描述 例子及解释
color 设置文本颜色 color: red; 将文本颜色设置为红色
font-size 设置字体大小 font-size: 16px; 将字体大小设置为 16 像素
font-family 设置字体系列 font-family: Arial, sans-serif; 将字体设置为 Arial，如果不可用，则使用默认的 sans-serif 字体系列
font-weight 设置字体粗细 font-weight: bold; 将字体设置为粗体
text-align 设置文本对齐方式 text-align: center; 将文本内容水平居中对齐
text-decoration 设置文本装饰效果（如下划线、删除线等） text-decoration: underline; 将文本添加下划线
background-color 设置背景颜色 background-color: #f1f1f1; 将背景颜色设置为十六进制颜色码"#f1f1f1"
background-image 设置背景图像 background-image: url("image.jpg"); 使用名为"image.jpg"的图像作为背景图像
padding 设置元素内边距（空白区域） padding: 10px; 将元素的内边距设置为 10 像素
margin 设置元素外边距（元素与其他元素之间的空白区域） margin: 20px; 将元素的外边距设置为 20 像素
border 设置元素边框 border: 1px solid black; 将元素的边框设置为 1 像素宽的黑色实线
width 设置元素宽度 width: 200px; 将元素宽度设置为 200 像素
height 设置元素高度 height: 150px; 将元素高度设置为 150 像素
display 设置元素的显示方式 display: flex; 将元素的显示方式设置为弹性布局
position 设置元素的定位方式 position: absolute; 将元素的定位方式设置为绝对定位
top, right, bottom, left 设置元素相对于其定位父元素的位置 top: 10px; 将元素相对于其定位父元素顶部偏移 10 像素
opacity 设置元素的不透明度 opacity: 0.5; 将元素的不透明度设置为 50%
box-shadow 设置元素的阴影效果 box-shadow: 2px 2px 4px rgba(0, 0, 0, 0.2); 添加一个 2 像素偏移、模糊半径为 4 像素的、颜色为 rgba(0, 0, 0, 0.2)的阴影效果
transition 设置元素的过渡效果 transition: width 0.3s ease-in-out; 当元素的宽度发生变化时，以 0.3 秒的时间以缓入缓出的方式进行过渡效果
transform 设置元素的变换效果（如旋转、缩放等） transform: rotate(45deg); 将元素旋转 45 度

# 4. 盒子模型

盒子模型是用于描述网页中元素布局和渲染的一种模型。每个元素都被看作是一个矩形的盒子，这个盒子包含内容、内边距、边框和外边距等部分。了解盒子模型对于进行网页布局和样式设计非常重要。

盒子模型由以下几个部分组成：

  1. 内容（content）：盒子中实际显示的内容，例如文本、图像等。
  2. 内边距（padding）：内容与边框之间的空白区域，用于控制内容与边框的距离。
  3. 边框（border）：内边距外部的边框线，用于围绕内容和内边距。
  4. 外边距（margin）：边框外部的空白区域，用于控制盒子与其他元素之间的距离。

在CSS中，可以使用相应的属性来控制盒子模型的不同部分。

下面是一个示例代码，展示了如何使用CSS来设置盒子模型的属性：

<!DOCTYPE html>
<html>
<head>
  <!-- CSS 样式 -->
  <style>
    .box {
      width: 200px;
      height: 200px;
      padding: 20px;
      border: 1px solid black;
      margin: 20px;
    }
  </style>
</head>
<body>
  <!-- html 标签结构 -->
  <div class="box">
    This is the content of the box.
  </div>
</body>
</html>

在上面的示例中，我们创建了一个类名为 "box" 的 <div>元素，并为其应用了一些盒子模型的属性。

具体解释如下：

  1. width 和 height 属性设置了盒子的宽度和高度为 200 像素。
  2. padding 属性设置了内边距为 20 像素，即内容与边框之间的空白区域为 20 像素。
  3. border 属性设置了边框样式为 1 像素宽的黑色实线边框。
  4. margin 属性设置了外边距为 20 像素，即盒子与其他元素之间的空白区域为 20 像素。

通过这些属性的组合，我们定义了一个具有固定大小、带有内边距和边框的盒子，并设置了它与其他元素之间的间距。

    background-color: #f1f1f1;
  }
  </style>
</head>
<body>
  <div class="container">
    <div class="box">

    </div>

  1. 左图右文布局

这个示例展示了一个左边是图片，右边是文本的布局。通过使用 Flexbox，我们将容器分为两个部分，图片和文本。图片部分占据了一定的宽度，并留出一些间距，文本部分则占据剩余的空间。

<!DOCTYPE html>
<html>
<head>
<style>
  .container {
    display: flex;
    align-items: center;
    width: 500px;
    border: 1px solid #e5e5e5;
  }
  .image {
    flex: 1;
    margin-right: 20px;
    width: 200px;
    display: flex;
  }
  .image img {
    max-width: 200px;
  }
  .text {
    flex: 1;
  }
  .text h2 {
    margin-top: 0;
  }
  .text p {
    margin-bottom: 0;
  }
</style>
</head>
<body>
<div class="container">
  <div class="image">
    <img src="image.jpg" alt="Image">
  </div>
  <div class="text">
    <h2>Title</h2>
    <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed nec ligula sed libero hendrerit efficitur.</p>

</div>
</body>

    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-gap: 20px;

  .item {
    text-align: center;
  }

  .item img {
    width: 300px;
  }

  .item h3 {
    margin-top: 0;
  }

  .item p {
    margin-bottom: 0;
  }

  .item .price {
    font-weight: bold;

</style>
</head>

<div class="container">

    <p>Description of Product 1</p>
