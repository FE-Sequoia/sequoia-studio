# 项目介绍

该项目是给客户定制系统的管理后台，整个项目分为三部分，管理后台、前台、移动端(同时兼容h5/微信小程序/app)。

客户要求整个项目都要用Vue3技术栈开发，故本项目作为一个Vue3技术栈的样板案例做一下介绍。

与Vue2相比，Vue3写出来的代码可以更加简洁并且复用性会更好。因为后台管理系统中的界面比较多的是表格列表和表单编辑数据，所以在项目开发中应用了Vue3的 useHooks 属性，开发了 useTable 和 useForm 两个 hooks 模块，用来提取表格和表单的公共方法。结合对应的页面组件，可以用极短的时间开发一个功能页面出来，大大提升了开发效率。

在左侧菜单部分采用了读取router.js路由配置文件信息，动态生成路由侧边菜单的设计思路，只需要维护路由文件就可以控制菜单与权限相关的东西，节省了工作量同时也提供了代码的维护性。

同时使用 vue-router 路由守卫的 beforeEach 方法对登录前后的跳转做了处理，限制用户登录前后的访问行为。比如登录后不允许访问登录页面，登录前不允许访问系统内部页面等。
